# Cookie-Shop-Q1-Analysis
"Financial performance analysis for The Cookie Shop (Q1). Includes data summary, expense tracking, and visualization of revenue vs. expenses."
## Project Overview
This repository contains the financial analysis for "The Cookie Shop" during the first quarter. This project demonstrates skills in data organization, financial calculation, and visualization.

## Key Metrics Analyzed
- *Total Revenue:* Tracking monthly sales performance.
- *Expense Tracking:* Analyzing costs across advertising, baking supplies, salaries, and utilities.
- *Profitability:* Calculation of net profit/loss to assess business health.

## Insights
- Identified peak revenue performance in January.
- Evaluated expense fluctuations to understand cost drivers.

## Tools Used
- *Microsoft Excel:* Data entry, formulas, and chart generation.
